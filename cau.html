Computer Systems Comprehensive Revision

1. CPU & Architecture

Von Neumann Architecture: The basic structure where data and instructions are stored in the same memory.

  • Clock Speed: Measured in Hertz (GHz). Number of cycles per second.
  • ALU (Arithmetic Logic Unit): Performs calculations and logic.
  • Registers: PC (holds next address), MAR (memory address), MDR (memory data), CIR (current instruction), Accumulator (results).
  • Buses: Address (one-way), Data (two-way), Control (commands).
  • Cache: Fast, small memory near CPU to reduce latency.

2. Hardware & Storage

  • RAM vs ROM: RAM is volatile (lost on power off); ROM is non-volatile (stores bootup).
  • Primary: RAM, Cache. Secondary: Hard Disk (Magnetic), SSD (Flash), Optical (CD/DVD).
  • Access: Sequential (Tape) vs Direct (HDD/SSD).
  • GPU: Specialized for graphics/parallel processing.

3. I/O & Peripherals

  • Printers: Inkjet (liquid ink), Laser (toner/heat), 3D (additive), Dot Matrix (impact), Plotter (vectors).
  • Input: Barcode (1D lines) vs QR Code (2D squares), Biometrics (fingerprint), Smart Cards (chips).
  • Display: Pixel Density (PPI), Color Depth (bits per pixel), Resolution ($W \times H$).
  • Ports: HDMI (AV), DVI (Video), VGA (Analog), DisplayPort.

4. Software & OS

  • System vs Application: OS/Drivers vs Word/Games.
  • OS Functions: Memory management, UI (GUI vs CLI), Utility (Defragmentation, Backup).
  • Processing: Batch (groups), Real-time (instant), Parallel (multi-core), Distributed (multi-computer).
  • Virtualization: Running multiple virtual systems on one physical hardware.
